# Repo-Level Autonomous Program Repair

## Introduction
Up to 2025-01-04, autonomous program repair technologies evaluated on SWE-Bench can be mainly divided into 3 catagories: Agent-Based Method, Agentless Based Method, and Fine-Tuned Based Method.
The workflow of these methods usually includes 3 phases: localization, repair generation, and repair validation.
